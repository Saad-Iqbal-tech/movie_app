# 🎬 Movie Explorer - Flutter App

A Flutter-based movie discovery app powered by the TMDB API.  
Browse movies, watch trailers, explore genres, and get recommendations in a smooth, animated UI.

---

## ✨ Features

- 🎥 Browse trending, popular, and top-rated movies  
- 🔎 Search movies in real-time  
- 🎭 Filter movies by genres  
- 🎞 Watch official trailers inside the app  
- 🤖 View recommended and similar movies  
- 🎨 Smooth animations and modern UI  
- 🖼 Clean movie grid and detail screens  
- ⚡ Fast image loading with caching
